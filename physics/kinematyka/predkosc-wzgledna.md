# Prędkość względna
Obliczamy ją, gdy ciała poruszają się względem siebie.
* Mijanie: kierunek taki sam, zwroty przeciwne

$V_{wzgl} = v_1 + v_2$

* Wyprzedzanie: kierunek taki sam, zwroty takie same

$V_{wzgl} = v_1 - v_2$
